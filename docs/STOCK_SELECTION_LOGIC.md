# 股票清單選擇邏輯說明

## 📊 概述

本系統的的股票清單**不是隨機挑選**，而是基於以下邏輯：

---

## 🎯 選擇標準

### 1. 市值排名 (Market Cap) - **主要標準**
- 選擇各市場市值最大的公司
- 確保流動性和穩定性
- 降低操縱風險

### 2. 指數成份股 (Index Components) - **核心依據**
| 市場 | 參考指數 | 成份股數量 |
|------|---------|-----------|
| **TW** | 台灣 50 (TAIEX 50) | 50 檔 |
| **CN** | 滬深 300 (CSI 300) | 300 檔 |
| **US** | S&P 300 | 300 檔 |

### 3. 交易量/流動性 (Volume/Liquidity)
- 日均交易量 > 一定門檻
- 確保進出場順暢
- 避免流動性風險

### 4. 產業代表性 (Sector Representation)
- 涵蓋各主要產業
- 分散產業風險
- 反映經濟結構

## 🔄 更新機制

### 定期檢視 (每季度)
- [ ] 檢查指數成份股調整
- [ ] 移除退市股票
- [ ] 新增新上市龍頭
- [ ] 調整產業權重

### 觸發條件
- **市值排名變化**: 跌出前 80% → 考慮移除
- **流動性下降**: 日均量 < 門檻 → 考慮移除
- **產業變化**: 新興產業崛起 → 考慮新增

## 📊 與隨機挑選的差異

| 特性 | 我們的邏輯 | 隨機挑選 |
|------|-----------|---------|
| **市值** | 大市值為主 | 大小混合 |
| **流動性** | 高流動性 | 不一定 |
| **產業** | 均衡分佈 | 可能集中 |
| **穩定性** | 高（指數成份） | 低 |
| **可解釋性** | 清晰規則 | 無規則 |
| **可複製性** | 完全可複製 | 不可複製 |

---

## 🎯 為什麼這樣選擇？

### 優點
1. **穩定性高**: 大市值公司較穩定
2. **流動性好**: 進出場容易
3. **數據品質**: 財報透明度高
4. **產業分散**: 降低單一產業風險
5. **可解釋性**: 規則清晰，可追蹤

### 缺點
1. **成長性限制**: 可能錯過小型成長股
2. **同質性**: 與大盤走勢高度相關
3. **調整延遲**: 指數調整有時間差
